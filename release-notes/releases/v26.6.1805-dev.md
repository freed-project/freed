## Freed v26.6.1805-dev

Dev build 1805

### Fixes

- Moved the sync trigger pickup into the native desktop process so backgrounded dev-channel builds can start Facebook, Instagram, and LinkedIn soaks from the terminal even when WebKit has suspended the renderer
- Installed dev-channel builds now install the sync trigger bridge as soon as the renderer bundle starts, so terminal Facebook, Instagram, and LinkedIn soaks no longer fail before background services finish initializing
- The native trigger watcher now waits and retries while the main renderer or sync bridge is unavailable, then reports a timeout instead of leaving an unattended run stuck
- Nightly self-improvement parsing now treats the explicit bug-scan outcome as the source of truth, so background runs can continue instead of misreading unfinished output
- Terminal sync triggers now verify Desktop consent and initialize the app store before invoking provider refresh, so hidden installed builds use persisted Facebook, Instagram, and LinkedIn auth instead of timing out on a React startup flag
- Terminal Facebook, Instagram, and LinkedIn sync triggers now fail zero-post, deferred, paused, or unauthenticated runs instead of reporting false success
- Native trigger timeouts now ignore stale requests, so an older hung renderer cannot overwrite the result for a newer terminal sync
- Sync trigger output now includes provider outcome details, post counts, added-item counts, and failure stage data so unattended runs can tell whether Facebook actually returned posts
- Internal unattended-run guidance now requires generated nightly plans, release soak notes, and handoff prompts to use terminal triggers or the 10 minute timeout path instead of waiting overnight on app clicks
- Terminal sync trigger requests now expire after the helper timeout, so old request files do not replay authenticated Facebook, Instagram, or LinkedIn traffic on the next launch
- Idle desktop memory recovery now recycles an old hidden main WebKit renderer when it is retaining reclaimable multi-GB RSS, without forcing Freed Desktop into focus
