## Freed v26.5.903-dev

Improves dense Friends graph recovery, Map interaction smoothness, desktop crash recovery, and WebKit memory attribution for today's dev builds

- Friends graph Fit all now waits for the first dense layout to finish before applying, so large graphs no longer get stuck off-screen when the action fires early

- Show crash recovery when rendering fails
